# 🎨 Семейная галерея

Интерактивный сайт для семейной игры с художниками.

## Структура проекта

```
├── index.html      ← главная страница (сайт)
├── data.js         ← все тексты про художников
└── images/         ← все картины (добавь сюда свои)
    ├── trio1_banksy_01.jpg
    ├── trio1_rerikh_01.jpg
    └── ...
```

## Как запустить на GitHub Pages

### 1. Создай репозиторий
- Зайди на github.com
- Нажми "New repository"
- Назови его `gallery` (или любое имя)
- Поставь **Public**
- Нажми "Create repository"

### 2. Загрузи файлы
Самый простой способ — через браузер:
- Открой репозиторий
- Нажми "uploading an existing file" или просто перетащи файлы
- Загрузи `index.html`, `data.js`
- Создай папку `images` и загрузи все картины

### 3. Включи GitHub Pages
- В репозитории зайди в **Settings** → **Pages**
- В разделе "Source" выбери: **Deploy from a branch**
- Branch: **main**, папка: **/ (root)**
- Нажми Save

### 4. Готово!
Через 1–2 минуты сайт будет доступен по адресу:
```
https://ТВО_ИМЯ.github.io/gallery/
```

---

## Как добавить новые тройки

Открой файл `data.js` и добавь новый объект в массив `trios` по образцу существующих.

Для картин — положи их в папку `images/` с именами вида:
```
trio9_художник_01.jpg
trio9_художник_02.jpg
```

И обнови ссылки в `data.js`.

---

## Управление на проекторе

- **Клик на картину** → полноэкранный режим
- **Стрелки на клавиатуре** → следующая/предыдущая картина в полноэкранном режиме
- **Esc** → выйти из полноэкранного режима
